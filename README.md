# ImageProcessing
Implementing Image Processing techniques.

# Implemented Algorithms (located in TargaImage.cpp)
- Uniform Quantization
- Populosity
- Naive Threshold Dithering
- Brightness Preserving Threshold
- Random Dithering
- Clustered Dithering
- Floyd-Steinberg Dithering
- Box Filter
- Bartlett Filter
- Gaussian Filter
- Compositing Over
- Compositing Inside
- Compositing Outside
- Compositing Atop
- Compositing Xor
